Talking points page

This demo shows how a support agent could explain denials in a clear, compliant, and empathetic way. The experience is fully local, so it avoids AWS costs and API keys while still feeling like a real operations tool.

  • Chat suggestions update from policy rules and case context.
  • Email triage classifies denial requests by reason and urgency.
  • Tasks are routed to the right queue with no backend dependency.
