# Free Contact Center Demo

This is a fully local, zero-cost demo built for a manager presentation.

## What it shows
- Customer chat with suggested agent replies
- Email triage with routing actions
- Task queue simulation
- Manager summary explaining the business value

## Why it is free
- No AWS account needed
- No paid APIs
- No external libraries
- Runs directly in the browser

## How to open
1. Open `index.html` in any browser.
2. Or use VS Code Live Server if you already have it installed.

## Demo tip
Walk through the scenario buttons at the top, then send a few messages in chat. Use the email and task action buttons to show routing behavior.
